# Sales Performance Analysis

## Project Overview

This project analyzes sales performance using the Superstore Sales Dataset. The goal is to identify key business insights, evaluate sales trends, measure profitability, and provide data-driven recommendations for improving business performance.

---

## Objectives

- Analyze overall sales and profit performance
- Calculate key business KPIs
- Identify top-performing and low-performing products
- Evaluate regional and category-wise performance
- Analyze monthly sales trends and seasonality
- Generate actionable business recommendations

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

The dataset contains information related to:

- Orders
- Customers
- Products
- Categories
- Regions
- Sales
- Profit

---

## Key Performance Indicators (KPIs)

The following KPIs were calculated:

- Total Revenue
- Total Profit
- Total Orders
- Average Order Value (AOV)
- Profit Margin (%)

---

## Analysis Performed

### 1. Data Cleaning
- Removed duplicate records
- Converted date columns to datetime format
- Checked missing values

### 2. Regional Performance Analysis
- Sales by Region
- Profit by Region

### 3. Category Analysis
- Category-wise Sales
- Category-wise Profit

### 4. Product Analysis
- Top 10 Products by Sales
- Worst Performing Products by Profit

### 5. Time Series Analysis
- Monthly Sales Trend
- Seasonal Sales Pattern

### 6. Customer Segment Analysis
- Segment-wise Sales
- Segment-wise Profit

## Business Recommendations

1. Focus marketing efforts on high-performing regions.
2. Promote top-selling products through targeted campaigns.
3. Review pricing strategies for low-profit products.
4. Improve inventory planning during peak seasons.
5. Strengthen customer retention programs for valuable segments.
